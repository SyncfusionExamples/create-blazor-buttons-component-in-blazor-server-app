### Syncfusion Blazor Buttons Component

This package contains the [Blazor Buttons Component](https://www.syncfusion.com/blazor-components/blazor-button?utm_source=github&utm_medium=listing&utm_campaign=blazor-lists-github) for Blazor application. The Blazor Buttons Component is a list-like interface that allows you to select an item or multiple items. It represents data in an interactive hierarchical structure across different layouts or views.
